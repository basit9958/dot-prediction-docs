## Installation

<CodeGroup>
```bash npm
npm install @dot-prediction/sdk
```
```bash yarn
yarn add @dot-prediction/sdk
```
</CodeGroup>

## Quick Start

```typescript
import { DotPredictionClient } from '@dot-prediction/sdk';
const client = new DotPredictionClient({ apiKey: 'your-api-key' });
const { markets } = await client.markets.list({ provider: 'polymarket', limit: '10' });
```

## Modules

| Module | Description |
|--------|-------------|
| `client.markets` | List, search, orderbooks, trades, price history |
| `client.events` | Events (groups of related markets) |
| `client.series` | Series (groups of related events) |
| `client.profile` | User profiles, trades, PnL, positions, activity |
| `client.system` | Health checks and platform capabilities |

## Pagination

Loop until `has_more` is `false`:

```typescript
let cursor: string | undefined;
do {
  const r = await client.markets.list({ provider: 'polymarket', limit: '50', cursor });
  cursor = r.pagination.has_more ? r.pagination.next_cursor ?? undefined : undefined;
} while (cursor);
```

## Errors

The SDK throws `"API Error: <msg>"` for 4xx/5xx, `"Request failed: <msg>"` for network errors.
