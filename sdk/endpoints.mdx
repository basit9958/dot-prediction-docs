## Markets

| Method | Description |
|--------|-------------|
| `client.markets.list(params)` | List markets with optional filtering |
| `client.markets.getById(params)` | Get a specific market by ID |
| `client.markets.getOrderbook(params)` | Get bid/ask depth for a market |
| `client.markets.getTrades(params)` | Get trade history for a market |
| `client.markets.getPriceHistory(params)` | Get OHLC price history |

### Example: List Markets

```typescript
const { markets, pagination } = await client.markets.list({
  provider: 'polymarket',
  limit: '20',
  status: 'active',
  query: 'bitcoin',
});
```

### Example: Orderbook

```typescript
const orderbook = await client.markets.getOrderbook({
  provider: 'polymarket',
  id: 'market-id',
});
// orderbook.data.yes.bids -> [{ price, size }]
```

### Example: Price History

```typescript
const history = await client.markets.getPriceHistory({
  provider: 'polymarket',
  market_ids: 'id-1,id-2',
  start_ts: 1700000000,
  end_ts: 1710000000,
  interval: 60,
});
```

## Profile

| Method | Description |
|--------|-------------|
| `client.profile.getInfo(params)` | User profile and trading metrics |
| `client.profile.getTrades(params)` | Paginated user trades |
| `client.profile.getPnL(params)` | PnL over time |
| `client.profile.getPositions(params)` | Active positions |
| `client.profile.getActivity(params)` | Merge/split/redeem activity |

### Example: PnL

```typescript
const pnl = await client.profile.getPnL({
  provider: 'polymarket',
  user: '0x1234...',
  granularity: 'week',
});
// pnl.pnl_over_time -> [{ timestamp, pnl, cumulative_pnl }]
```
