## What is Dot Prediction?

Dot Prediction is a prediction market data aggregation API. Instead of integrating with each platform's API individually, you can use Dot Prediction to query normalized data from all supported platforms through a single endpoint.

## Supported platforms

| Platform | Markets | Events | Series | Trades | Order Book | Price History | Profiles |
|----------|---------|--------|--------|--------|------------|---------------|----------|
| **Kalshi** | Yes | Yes | Yes | Yes | Yes | Yes | Yes |
| **Polymarket** | Yes | Yes | Yes | Yes | Yes | Yes | Yes |
| **Limitless** | Yes | No | No | No | Yes | Yes | No |

## Key features

- **Unified interface** - One API, multiple prediction market platforms
- **Normalized data** - Consistent response formats regardless of the source platform
- **Real-time data** - Markets, order books, and trades updated in real time
- **OHLC price history** - Candlestick data at 1-minute, 1-hour, or 1-day intervals
- **User profiles** - Look up trader profiles and performance metrics
- **Cursor-based pagination** - Efficiently page through large result sets
