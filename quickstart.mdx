## Get started in three steps

<Steps>

<Step title="Create an account">
Register for a Dot Prediction account using email and password.

```bash
curl -X POST https://api.dotprediction.io/v1/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Your Name",
    "email": "you@example.com",
    "password": "your-password"
  }'
```

You can also sign up with [Google OAuth](/guides/authentication#google-oauth).
</Step>

<Step title="Generate an API key">
Once logged in, create an API key. You'll need the session cookie from registration or login.

```bash
curl -X POST https://api.dotprediction.io/v1/api/auth/api-keys \
  -H "Content-Type: application/json" \
  -b "jwt=YOUR_SESSION_COOKIE" \
  -d '{
    "name": "my-first-key"
  }'
```

The response includes your API key in the `key` field. Save it securely â€” you won't be able to retrieve it again.
</Step>

<Step title="Make your first request">
Use your API key to fetch prediction markets.

<CodeGroup>

```bash curl
curl https://api.dotprediction.io/v1/api/markets?provider=polymarket&limit=5 \
  -H "X-API-Key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api.dotprediction.io/v1/api/markets",
    params={"provider": "polymarket", "limit": 5},
    headers={"X-API-Key": "YOUR_API_KEY"}
)

markets = response.json()
for market in markets["markets"]:
    print(f"{market['title']} - {market['status']}")
```

```javascript JavaScript
const response = await fetch(
  "https://api.dotprediction.io/v1/api/markets?provider=polymarket&limit=5",
  { headers: { "X-API-Key": "YOUR_API_KEY" } }
);

const data = await response.json();
data.markets.forEach(market => {
  console.log(`${market.title} - ${market.status}`);
});
```

</CodeGroup>

</Step>

</Steps>
