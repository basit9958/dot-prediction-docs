## Base URL

All API requests use the following base URL:

```
https://api.dotprediction.io
```

## Authentication

All data endpoints require an API key passed via the `X-API-Key` header:

```bash
curl https://api.dotprediction.io/v1/api/markets?provider=polymarket \
  -H "X-API-Key: YOUR_API_KEY"
```

You can also use `Authorization: Bearer YOUR_API_KEY`. See the [Authentication guide](/guides/authentication) for details on obtaining an API key.

## Response format

Successful responses return data with pagination and metadata:

```json
{
  "pagination": {
    "total": 100,
    "limit": 20,
    "has_more": true,
    "next_cursor": "abc123"
  },
  "markets": [...],
  "meta": {
    "platforms_queried": ["polymarket"],
    "request_time": 45,
    "cache_hit": false,
    "data_freshness": "2025-01-15T12:00:00Z"
  }
}
```

Errors return a structured error object:

```json
{
  "error": {
    "status": 400,
    "message": "invalid provider",
    "resolution": "provider must be one of: kalshi, polymarket, limitless"
  }
}
```

## Rate limits

- **Free tier:** 250 requests/minute
- **Pro tier:** 500 requests/minute

See [Rate limiting](/guides/rate-limiting) for details.
